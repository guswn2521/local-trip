# 국내여행 모바일 관광가이드북

이번 여행 전용 정적 웹앱입니다. 관광지, 숙소, 식당, 카페 후보는 모두 `data.js`에 미리 등록된 고정 데이터만 사용합니다.

## 열어보기

브라우저에서 `index.html`을 직접 열어도 됩니다. 로컬 서버로 볼 때는 아래 명령을 사용하세요.

```bash
python3 -m http.server 4173
```

그 다음 `http://localhost:4173`을 엽니다.

## 범위

- 길찾기 기능 없음
- 실시간 검색 기능 없음
- 예약/영업시간 자동 확인 없음
- 모바일 화면 우선
- 운영시간/주차장은 미리 적어둔 참고 정보이며 방문 전 재확인이 필요
- 현재 영업 중/종료 상태는 표시하지 않고, 확인된 항목만 요일별 시간표 형태로 표시
- 2026-07-02 기준 네이버지도 상세 화면에서 확인된 주차 정보는 `data.js`의 `naverVerifiedDetails`에 별도 보관
- 주차요금이 확인된 장소는 카드 안에 차량 종류별 가격표로 표시
- 사찰·유적 안의 석탑처럼 하위 명소인 경우 `naverQuery`로 대표 장소 검색 링크를 따로 지정
- 대표사진은 실제 출처를 확인한 후보에만 표시
- 후보 목록은 목차와 가로 카드 넘김 방식으로 탐색
- 관광지, 숙소, 식당, 카페 버튼은 복수 선택 토글이며 선택한 종류를 한 지도에 함께 표시
- 목차 안에서도 관광지, 숙소, 식당, 카페 탭으로 후보를 구분해서 표시
- `전체 보기` 버튼은 관광지, 숙소, 식당, 카페 탭별로 이름, 지역, 운영시간, 주차 가능 여부 목록을 표시
- 각 후보 카드에서 네이버지도 영업시간·주차 정보를 바로 확인 가능
